\# News Orchestrator



This project fetches news articles, summarizes them, and displays them in a structured format.



\## Features



\- Fetch latest news using News API

\- Summarize articles using OpenAI API

\- Configurable number of articles

\- Easy to extend modules



\## Setup



1\. Install dependencies:

```bash

pip install -r requirements.txt



2\. Add .env file with your API keys:



NEWS\_API\_KEY=your\_news\_api\_key

OPENAI\_API\_KEY=your\_openai\_api\_key



3\. Run the app:



python app.py



Save and close the file.
